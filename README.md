# uni-app

个人 uni-app 项目（Vue3 + TypeScript）。

## 技术栈

- uni-app
- Vue 3
- TypeScript
- Pinia
- uni-ui

## 安装依赖

```bash
pnpm i --registry=https://registry.npmmirror.com
```

## 本地开发

```bash
# H5
pnpm run dev:h5

# 微信小程序
pnpm run dev:mp-weixin
```

## 打包

```bash
# H5
pnpm run build:h5

# 微信小程序
pnpm run build:mp-weixin
```
